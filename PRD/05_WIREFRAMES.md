## 키보드 단축키 맵

| 단축키 | 기능 | Phase |
|--------|------|-------|
| `Ctrl+K` | 커맨드 팔레트 (검색/이동/명령) | P1 |
| `Ctrl+F` | 현재 결과 내 필터 | P1 |
| `Ctrl+B` | 현재 저장소 북마크 | P1 |
| `Ctrl+,` | 설정 열기 | P1 |
| `Ctrl+1~3` | 저장소/코드/이슈 탭 전환 | P1 |
| `Enter` | 선택한 저장소 상세 보기 | P1 |
| `Ctrl+Enter` | 선택한 저장소 GitHub에서 열기 | P1 |
| `↑/↓` | 검색 결과 탐색 | P1 |
| `Ctrl+Shift+C` | 유사 저장소 비교 | P2 |
| `Ctrl+Q` | 코드 Q&A 열기 | P2 |
| `?` | 단축키 도움말 | P1 |
