## Phase 1: 기반 + 핵심 검색 (MVP 코어)

### 목표
Phase 1이 끝나면 자연어로 GitHub 저장소/코드/이슈를 검색하고, AI 요약으로 결과를 빠르게 파악하고, 키보드 단축키로 빠르게 조작할 수 있다.

### 기능
- [ ] Tauri 2.0 데스크톱 앱 셸 + React UI 기반 구축
- [ ] GitHub OAuth 로그인 (API 접근 권한 획득)
- [ ] GitHub API 연동 (저장소/코드/이슈 검색)
- [ ] AI 시맨틱 통합 검색 (클라우드 AI로 의미 기반 매칭)
- [ ] 검색 결과 AI 요약 (상단에 "이런 것들을 찾았어요" 1줄 요약)
- [ ] 검색 결과 카드 UI (저장소/코드/이슈 탭 분리)
- [ ] 검색 히스토리 저장 및 재검색
- [ ] 키보드 단축키 시스템 (Ctrl+K 검색, 커맨드 팔레트)
- [ ] SQLite 로컬 DB 설정
- [ ] 기본 설정 화면 (GitHub 계정, AI API 키 관리)
- [ ] 다크모드 / 라이트모드 UI 테마

### 데이터
- Repository, SearchHistory, AIProvider

### 인증
- GitHub OAuth App (github.com/settings/developers에서 등록)

### "진짜 제품" 체크리스트
- [ ] 실제 GitHub API 연결 (목업 데이터 X)
- [ ] 실제 AI API 연결 (하드코딩된 응답 X)
- [ ] 실제 SQLite DB 사용 (메모리 저장 X)
- [ ] 설치 파일(.msi/.dmg/.deb) 생성 가능
- [ ] 키보드만으로 검색 → 결과 탐색 가능

## Phase 2: 코드 인텔리전스 + 컬렉션 + 분석

### 전제 조건
- Phase 1이 안정적으로 동작하는 상태

### 목표
Phase 2가 끝나면 저장소 전체 코드에 대해 자연어로 질문하고, 컬렉션+스마트 폴더로 저장소를 관리하고, 건강도 분석과 비교로 최적의 저장소를 고를 수 있다.

### 기능
- [ ] 자연어 질문 → 코드 답변 (RAG 기반, 멀티파일 지원)
- [ ] 벡터 임베딩 캐시 시스템 (sqlite-vss, 모델별 자동 차원 처리)
- [ ] 북마크 & 컬렉션 관리 UI (생성/삭제/이동/메모)
- [ ] 컬렉션 스마트 폴더 (자동 분류 규칙: 언어+스타 수 조건)
- [ ] 저장소 건강도 분석 (스타/커밋빈도/이슈응답/라이선스 → 점수)
- [ ] 유사 저장소 비교 (2~3개 나란히 비교표 자동 생성)
- [ ] AI 대화 기록 저장 & 검색 (무제한 보관)
- [ ] 코드 스니펫 뷰어 (구문 하이라이트)
- [ ] 고급 필터 (언어, 스타 수, 토픽, 최근 업데이트)

### 추가 데이터
- Collection, CollectionItem, Embedding, Conversation, Message
- Repository에 health_score 필드 추가

### 통합 테스트
- Phase 1 검색 기능이 여전히 정상 동작하는지 확인
- 검색 → 건강도 확인 → 비교 → 북마크 → 코드 질문 전체 흐름 테스트

## Phase 3: 로컬 AI + 스마트 추천 + 트렌딩

### 전제 조건
- Phase 1 + 2가 안정적으로 동작하는 상태

### 목표
Ollama로 오프라인 검색이 가능하고, 내 취향을 학습한 AI가 저장소를 추천하고, 트렌딩 대시보드로 매일 새로운 발견이 있는 완성도 높은 앱.

### 기능
- [ ] Ollama 로컬 AI 연동 (설치된 모델 자동 감지)
- [ ] 오프라인 모드 (인터넷 없이 캐시된 데이터로 검색)
- [ ] AI 모델 선택 UI (GPT/Claude/Ollama 전환)
- [ ] 스마트 추천 엔진 (검색/북마크 패턴 학습 → "이것도 좋아할 수 있어요")
- [ ] 트렌딩 대시보드 (GitHub 트렌딩을 AI가 요약, 매일 업데이트 뉴스피드)
- [ ] 컬렉션 내보내기/가져오기 (JSON)
- [ ] 성능 최적화 & 캐시 전략 (7일 자동 갱신)
- [ ] 자동 업데이트 (GitHub Releases + Tauri updater)

### 추가 데이터
- UserPreference (추천 학습용), TrendingSnapshot (트렌딩 기록)

### 주의사항
- Ollama 설치 필요 (사용자 PC에 별도 설치)
- 로컬 모델 성능은 PC 사양에 따라 차이남
- 오프라인 모드는 이전에 캐시된 데이터만 검색 가능
- 추천 엔진은 충분한 사용 데이터가 쌓여야 정확도 향상
