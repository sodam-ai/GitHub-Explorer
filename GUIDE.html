1. 이 프로그램이 뭐 하는 건가요

GitHub(깃허브)는 전 세계 개발자들이 만든 프로그램의 소스 코드(설계도라고 생각하시면 됩니다)를 올려두는 큰 창고 같은 웹사이트입니다. 그런데 이 창고에는 물건이 너무 많아서(수억 개), 원하는 걸 찾기가 쉽지 않습니다.

GitHub AI Explorer는 이 창고에서 원하는 걸 평소 말하듯이 찾아주는 프로그램입니다. 예를 들어 "React 드래그앤드롭 라이브러리"라고 입력하면, 그와 관련된 것들을 찾아줍니다. 검색엔진(예: 구글)에서 검색하는 것과 비슷하다고 생각하시면 됩니다.

  • 인터넷 사이트가 아니라 컴퓨터에 설치하는 프로그램입니다 (카카오톡, 크롬 같은 것처럼 내 컴퓨터에 설치해서 씁니다)
  • 처음엔 AI 기능 없이도 다 쓸 수 있고, 나중에 원하면 AI 기능도 켤 수 있습니다
  • 이 프로그램은 여러분의 개인정보를 어디로 보내지 않습니다

2. 시작하기 전에 준비할 것

준비물꼭 필요한가요?설명
Windows, Mac, 또는 Linux 컴퓨터예, 필수스마트폰이나 태블릿에서는 안 됩니다
인터넷 연결예, 필수와이파이나 랜선으로 인터넷이 연결되어 있어야 합니다
GitHub 계정아니요, 없어도 됩니다나중에 만들어도 됩니다

"내 컴퓨터가 Windows인지 Mac인지 모르겠어요" — 화면 왼쪽 아래(Windows) 또는 화면 위쪽(Mac)에 있는 로고 모양으로 구분할 수 있습니다. Windows는 네모 4개가 붙은 파란 창문 모양, Mac은 사과 모양 로고입니다.

3. 다운로드하는 방법 (아주 자세히)

"다운로드"란 인터넷에 있는 파일을 내 컴퓨터로 가져오는 것을 말합니다.

  1. 인터넷 브라우저(크롬, 엣지, 사파리 등 인터넷 하는 프로그램)를 엽니다
  2. 주소창(맨 위에 URL을 입력하는 칸)에 다음 주소를 입력하고 Enter를 누릅니다: https://github.com/sodam-ai/GitHub-Explorer
  3. 화면에서 "Releases"(릴리스)라고 써있는 곳을 찾아 클릭합니다 — 보통 화면 오른쪽에 있습니다
  4. 목록 중 가장 위에 있는 것(최신 버전)을 클릭합니다
  5. 내 컴퓨터 종류에 맞는 파일을 클릭해서 다운로드합니다
    • Windows를 쓰신다면 → 이름 끝이 .msi 또는 .exe로 끝나는 파일
    • Mac을 쓰신다면 → 이름 끝이 .dmg로 끝나는 파일
  6. 다운로드가 끝나면 보통 브라우저 화면 아래쪽이나, 내 컴퓨터의 "다운로드" 폴더에서 파일을 찾을 수 있습니다
⚠️ 알아두실 점: 이 문서를 쓰는 시점에 이 저장소에 실제로 다운로드할 수 있는 파일이 올라와 있는지는 위 3번 단계에서 직접 확인하셔야 합니다. 만약 "Releases"에 아무것도 없다면, 아직 완성된 배포 파일이 없는 상태일 수 있습니다 — 이 경우 개발자에게 문의가 필요합니다.

4. 설치하는 방법 (아주 자세히)

"설치"란 다운로드한 파일을 내 컴퓨터에서 실제로 쓸 수 있게 만드는 과정입니다.

  1. 앞서 다운로드한 파일을 찾아서 마우스로 두 번 빠르게 클릭(더블클릭)합니다
  2. 화면에 설치 안내 창이 뜹니다. 대부분 "다음", "동의함", "설치" 순서로 버튼을 누르면 됩니다
  3. Windows에서 파란색 경고 화면이 뜨는 경우: "Windows에서 PC를 보호했습니다"라는 문구가 보일 수 있습니다. 이건 위험하다는 뜻이 아니라, 아직 많이 알려지지 않은 프로그램이라 뜨는 표준 경고입니다. 화면의 "추가 정보"를 클릭한 뒤 "실행" 버튼을 누르면 계속 진행됩니다
  4. 설치가 끝나면 "완료" 버튼을 누릅니다
  5. 이제 컴퓨터의 프로그램 목록에서 "GitHub AI Explorer"를 찾을 수 있습니다
    • Windows: 화면 왼쪽 아래 시작 버튼 클릭 → 프로그램 이름 검색
    • Mac: 런치패드(로켓 모양 아이콘)에서 찾기

5. 처음 실행했을 때

프로그램을 처음 켜면 사용법을 짧게 안내하는 화면이 뜹니다.

  • "다음" 버튼을 누르면 예시를 하나씩 보여줍니다
  • "건너뛰기" 버튼을 누르면 바로 메인 화면으로 이동합니다

둘 중 아무거나 선택해도 괜찮습니다. 이 안내는 다시 보고 싶어도 자동으로는 다시 뜨지 않습니다(한 번 본 걸 기억합니다).

6. 기본 사용법 — 저장소 찾기

화면 가운데에 있는 입력 칸(검색창)을 클릭하고, 원하는 것을 문장으로 적어보세요.

예시 입력:

  • React 드래그앤드롭 라이브러리
  • Python으로 만든 크롤링 도구
  • 할일 관리 앱

입력 후 키보드의 Enter 키를 누르거나, 옆에 있는 "검색" 버튼을 클릭합니다.

결과가 뜨면:

  • 화면 위쪽에 "저장소" / "코드" / "이슈" 3개의 탭(칸막이)이 있습니다. "저장소"가 우리가 찾는 프로그램 하나하나를 말합니다.
  • 각 항목을 클릭하면 더 자세한 설명, 별(⭐, 사람들이 얼마나 좋아하는지 나타내는 표시), 최근 소식 등을 볼 수 있습니다

인터넷이 안 될 때는? 걱정하지 마세요. 예전에 검색했던 내용 안에서는 계속 찾아볼 수 있습니다. 다만 새로운 검색은 인터넷이 있어야 합니다.

7. 마음에 드는 것 저장하기 (북마크)

마음에 드는 저장소를 나중에 다시 보고 싶다면:

  1. 저장소 카드(목록에 보이는 항목)에 있는 책갈피 모양 아이콘을 클릭합니다
  2. 처음이라면 "새 폴더(컬렉션) 만들기"를 선택해서 이름을 지어줍니다(예: "나중에 볼 것")
  3. 메모를 남기고 싶으면 메모 칸에 적을 수 있습니다 (메모에 인터넷 주소를 적으면 자동으로 클릭 가능한 링크가 됩니다)
  4. 화면 위쪽 "컬렉션" 메뉴에서 저장해둔 것들을 모아볼 수 있습니다

8. AI 기능은 뭐고, 어떻게 켜나요

이 프로그램의 기본 검색 기능은 AI 없이도 다 됩니다. 그런데 "이 검색 결과를 요약해줘" 또는 "이 코드가 뭐 하는 건지 설명해줘" 같은 건 AI(인공지능)의 도움이 필요합니다.

AI를 쓰려면 "열쇠(API 키)"가 필요합니다

여기서 말하는 AI는 OpenAI(챗GPT를 만든 회사), Anthropic(Claude를 만든 회사), Google(Gemini) 같은 곳에서 제공하는 서비스입니다. 이런 서비스를 이용하려면 "API 키"라는, 긴 비밀번호 같은 문자열이 있어야 합니다. 이건 각 회사의 웹사이트에서 회원가입 후 발급받는 것으로, 이 프로그램이 대신 만들어주지 않습니다.

API 키가 없어도 되는 방법: 내 컴퓨터에 "Ollama"나 "LM Studio"라는 별도 프로그램을 설치하면, 그 안에서 AI가 내 컴퓨터 안에서만 작동합니다(인터넷도 필요 없고 키도 필요 없습니다). 다만 이 프로그램들을 따로 설치해야 하고, 컴퓨터 사양이 어느 정도 필요합니다.

설정 방법

  1. 프로그램 오른쪽 위 톱니바퀴 모양 아이콘(설정)을 클릭합니다
  2. "AI 제공자"라고 써있는 부분에서 원하는 서비스 이름 옆 칸에 API 키를 붙여넣습니다
  3. "저장" 버튼을 누릅니다
  4. 그 아래 있는 드롭다운(선택 상자)에서 실제로 사용할 서비스를 골라줍니다 — 이 단계를 빼먹으면 키를 넣어도 작동하지 않습니다

10. 문제가 생겼을 때

"실행이 안 돼요"
→ 설치가 제대로 됐는지 확인해보세요. 4번 항목을 다시 따라 해보세요.

"검색해도 결과가 안 나와요"
→ 인터넷 연결을 확인하세요. 와이파이가 켜져 있는지, 다른 사이트는 잘 열리는지 확인해보세요.

"AI 요약이 안 보여요"
→ 8번 항목을 참고해서 API 키를 넣고, 드롭다운에서 사용할 서비스를 선택했는지 확인하세요.

"자꾸 이상한 영어 메시지가 떠요"
→ 화면을 캡처(스크린샷)해서 기록해두세요. 원인 파악에 도움이 됩니다. 대부분은 인터넷 연결 문제나, AI 서비스 쪽 문제입니다.

그래도 해결이 안 되면?
→ 이 프로그램의 GitHub 페이지(https://github.com/sodam-ai/GitHub-Explorer)에서 "Issues"(이슈, 문제 보고) 메뉴를 통해 문제를 알릴 수 있습니다.

11. FAQ

Q. 돈을 내야 하나요?
A. 프로그램 자체는 무료입니다. 다만 AI 기능을 OpenAI 같은 유료 서비스로 쓰려면, 그 회사에 별도로 돈을 낼 수 있습니다. 로컬 AI(Ollama 등)를 쓰면 무료입니다.

Q. 제 정보가 새어나가나요?
A. 이 프로그램을 만든 사람에게 여러분의 정보가 전송되는 경로는 없습니다. 검색은 GitHub로, AI 질문은 여러분이 직접 선택한 AI 회사로만 전달됩니다.

Q. 스마트폰에서도 되나요?
A. 안 됩니다. 컴퓨터(Windows/Mac/Linux) 전용입니다.

Q. 실수로 삭제하면 어떻게 되나요?
A. 저장해둔 것들은 내 컴퓨터 안에 있어서, 프로그램을 지워도 다시 설치하면 대부분 남아있을 수 있습니다. 다만 확실히 하려면 "설정 → 데이터 → 컬렉션 내보내기"로 미리 백업 파일을 만들어두는 것을 추천합니다.

12. 사용해도 되는지 (라이선스)

이 부분은 법률 상담이 아닙니다. 정확한 판단이 필요하면 변호사와 상담하세요.

이 프로그램의 소스 코드(설계도)를 다른 곳에 가져다 쓰거나, 이걸로 돈을 버는 것(상업적 이용)은 지금 이 문서 작성 시점 기준으로 명확히 허락되어 있지 않습니다. 저장소에 사용 허락 문서(라이선스)가 아직 없기 때문입니다.

그냥 프로그램을 다운로드해서 개인적으로 쓰는 것은 이 문제와 상관없이 괜찮습니다. 문제가 되는 건 "소스 코드를 가져다가 내가 다른 프로그램을 만드는 데 쓰는 것" 같은 경우입니다. 이런 걸 하고 싶으시면 저장소를 만든 사람에게 먼저 물어보세요.
